# 🧠 Smart Study Planner (CLI)

## 📌 About the Project

This project is a simple **Command Line Interface (CLI)** application designed to help users organize their study schedule and create a daily study plan.

The user enters their courses, and the system manages and plans them.

---

## 🚀 Features

* 📚 Add courses
* 📋 List all courses
* ❌ Delete a course
* 🗓️ Generate a daily study plan
* 📊 Show the most studied course

---

## 🛠️ Concepts Used

This project is built to practice core Python concepts:

* `for` / `while` loops
* `break` / `continue`
* `range()`
* `enumerate()`
* `zip()`
* List Comprehension
* Functions
* `lambda`, `map`, `filter`

---

## 💻 How It Works

When the program starts, a menu is displayed:

1. Add Course
2. List Courses
3. Delete Course
4. Create Daily Plan
5. Show Most Studied Course
6. Exit

The program runs the corresponding function based on user input.

---

## ▶️ How to Run

Make sure Python is installed, then run:

```bash
python main.py
```
